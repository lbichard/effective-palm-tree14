## Installation
First, like always, type in the intergrated terminal "npm i". Then type:

```
    mysql -u root -p        Press Enter
```
- Then, enter the password.

- Then, type:

```
    source db/schema.sql    Press Enter
```

- Then, type:

```
    exit                    Press Enter
```

- Then, type:

```
    npm run seed            Press Enter
```
- Then, type:

```
    npm start               Press Enter
```
